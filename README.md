# Mi Tienda Online - E-commerce de Café

## Descripción
Sitio web de e-commerce especializado en la venta de cafés premium y accesorios para preparar café. La página ofrece una experiencia completa de compra online con carrito dinámico y formulario de contacto funcional.

## Características Principales
- **Catálogo de Productos**: Granos de café premium y accesorios para preparar café
- **Carrito de Compras**: Funcionalidad completa con localStorage para persistencia
- **API Integration**: Consumo de productos desde FakeStore API
- **Diseño Responsivo**: Adaptable a todos los dispositivos
- **Formulario de Contacto**: Integrado con Formspree para envío de mensajes

## Tecnologías Utilizadas
- HTML5 semántico
- CSS3 con Flexbox y Grid
- JavaScript ES6+
- Google Fonts
- Formspree para formularios
- FakeStore API para productos

## Funcionalidades
- ✅ Navegación suave entre secciones
- ✅ Carrito de compras con contador dinámico
- ✅ Persistencia de datos con localStorage
- ✅ Validación de formularios
- ✅ Diseño responsivo
- ✅ Optimización SEO y accesibilidad
- ✅ Animaciones y efectos visuales

## Instalación y Uso
1. Clonar o descargar el repositorio
2. Abrir `index.html` en un navegador web
3. La página es completamente funcional sin necesidad de servidor
